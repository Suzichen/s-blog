# `@spage/engine-win32-x64-msvc`

Platform-specific native binary for [`@spage/engine`](https://www.npmjs.com/package/@spage/engine) on Windows x64.

This package is installed automatically as an optional dependency — you should not need to install it directly.

See the [main package](https://www.npmjs.com/package/@spage/engine) for API documentation.
